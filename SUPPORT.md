# Mateo support

Use the issue chooser for help adopting Mateo, clarifying design guidance, or
reporting unexpected behavior. Include the affected package version or design
document and a small example whenever possible.

- Usage question or design clarification: open a Question issue.
- Reproducible defect: open a Bug report.
- New or expanded component behavior: use the matching component request.
- Security concern: follow [SECURITY.md](SECURITY.md) privately.

Mateo is maintained in public without a guaranteed response time. Clear,
focused reports are easier for maintainers and community members to answer.
