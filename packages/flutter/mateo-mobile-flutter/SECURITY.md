# Security policy

Mateo Mobile is a public UI package. Please report security issues privately so
they can be understood and fixed before details are shared publicly.

## Reporting a vulnerability

Email security reports to security@ventairy.com with:

- the affected package and version;
- a clear description of the issue;
- reproduction steps or a minimal example when possible;
- any known impact or affected platforms.

Do not open a public issue for a suspected vulnerability.

## Response

Security reports are reviewed privately. If the issue affects a released Mateo
package, the fix will ship through the package's normal release lane and the
release notes will describe the user-visible impact once disclosure is safe.
